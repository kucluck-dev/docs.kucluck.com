# Commands

import { Callout } from "nextra/components";

## Vehicle Parking
You can use the parking command to pick up or store a vehicle

## Delete All Vehicles
Use this command to delete all vehicles. Only players who have permissions can use this command. 
- You can change the default command in `shared/config.lua` line 26.
- You can also set permissions using the command at `shared/config.lua` line 56.
<Callout type="info">
  <p>Default command "asuransi"</p>
</Callout>

## Lock or Unlock Vehicles
You can change the default command in `shared/config.lua` line 27.
This command is integrated with keybind

## Give Vehicle Keys
This command functions to provide temporary vehicle keys.
