## getVehicleParkingFromPlate
Has a function to get vehicle parking data

**Usage:** 
```lua copy
exports['kc_garagev2']:getVehicleParkingFromPlate(plate)
```
| Code Block  | Value   | Explanation   |
| :-----      | :----:  | ----:         |
| plate       | string  | Vehicle plate |

- return: `table`
  - name: `string`,
  - label: `string`,
  - dateIn: `number`,
