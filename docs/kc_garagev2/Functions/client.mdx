## JobsImpound

Use this to impound a vehicle by a specific job

### Usage

```lua title="client.lua"
exports['kc_garagev2']:JobsImpound(impoundName, vehicle, identifier)
```

| Code Block  | Value  |                     Explanation |
| :---------- | :----: | ------------------------------: |
| impoundName | string |          Your impound code name |
| vehicle     | number | You can use GetVehiclePedIsIn() |
| identifier  | string |               Player identifier |

### Example

```lua title="client.lua"
RegisterCommand('JobsImpound', function()
  local playerCoords = GetEntityCoords(PlayerPedId())
  local vehicle = GetClosestVehicle(playerCoords, 5.0)
  local playerData = ESX.GetPlayerData()
  exports['kc_garagev2']:JobsImpound('JobsImpound', vehicle, playerData.identifier)
end)
```
