## getGarages

Use this to get all the garages data

### Usage

```lua copy
exports['kc_garagev2']:getGarages()
```

- return: `table[]`
  - You will get all the garages data.

## getGarageLabel

Has a function to get the garage label and coords of the garage name

### Usage

```lua copy
exports['kc_garagev2']:getGarageLabel(garageName)
```

| Code Block | Value  |          Explanation |
| :--------- | :----: | -------------------: |
| garageName | string | key name form garage |

- return:
  - label: `string`
  - coords: `vector4`
