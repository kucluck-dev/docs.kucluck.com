## Vehicle Parking

Use command /parking to spawn or store a vehicle

## Delete All Vehicles

Use this command to delete all vehicles. Only players who have permissions can use this command.<br/>
You can change the default command in `shared/config.lua` line 30.<br/>
You can also set permissions using the command at `shared/config.lua` line 40.

<Callout type="info">
  <p>Default command `impound`</p>
</Callout>

## Lock or Unlock Vehicles

You can turn off in `shared/config.lua` line 33 if can't use vehicle lock.<br/>
You can change the default command in `shared/config.lua` line 34.<br/>
This command is integrated with keybind in `shared/config.lua` line 35.

<Callout type="warn">
  <p>Turning off this if you're using `qb-vehiclekeys`</p>
</Callout>

## Give Vehicle Keys

This command functions to provide temporary vehicle keys.<br/>
You can change the default command in `shared/config.lua` line 36.

<Callout type="info">
  <p>Default command `givekeys`</p>
</Callout>

## Premium Garage

Use command `/premiumgarage` to buy premium garages.<br/>
Command can edited in `shared/config.lua` line 38.
